# Fork notice

This repository is a research fork of the official OpenUnlearning repository:

- Upstream repository: https://github.com/locuslab/open-unlearning
- Upstream project: OpenUnlearning, a unified framework for LLM unlearning benchmarks
- Upstream license: MIT

This fork contains additional code and runbooks used for a diploma project on
counterfactual and routed LLM unlearning. The main additions are DualCF v2
artifact preparation, rarity/difficulty/attribution scoring, DUET/RWKU production
runbooks, additional baselines, checkpoint/utility evaluation tooling, and result
packaging utilities.

Large model weights, local dataset mirrors, cache directories, and full training
outputs are not stored in this Git repository.
